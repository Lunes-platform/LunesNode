# Lunes Node

Lunes Node development

## Installation

### SBT (Scala Build Tool)

Install SBT [here](https://www.scala-sbt.org/1.0/docs/Setup.html).

### Run the project

Clone this repository and execute the command:

```
sbt "run lunes.conf"
```
